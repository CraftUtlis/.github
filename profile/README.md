# Welcome to CraftUtils

We are an open-source community dedicated to building powerful tools and utilities for the Minecraft ecosystem. Our mission is to bridge the gap between different game editions and simplify server management for everyone.

## Current Projects

### [LegacyCrossPlay](https://github.com/CraftUtlis/LegacyCrossPlay)
A specialized proxy translator that allows Minecraft Legacy Console Edition players to connect to 1.8 Java Edition servers. 

- **Status:** Active / Public
- **License:** GNU GPL v3.0
- **Key Features:** Protocol translation, Java server compatibility, easy setup.

---

## Planned & Upcoming

### Minecraft Server Dashboard (WIP)
A modern, web-based UI for Minecraft servers built with **React.js**. Designed to replace tedious manual file editing and console management with a sleek interface.

- **Status:** 💡 Planning / Development stage
- **Core Features:**
    - **Kernel Manager:** One-click download and installation of various server cores (Paper, Purpur, Velocity, etc.).
    - **Visual Configurator:** Full UI for editing `server.properties` and other config files without touching a text editor.
    - **Plugin Store:** Easy management and installation of plugins.
    - **Native Integration:** Built-in toggle to instantly enable and configure **LegacyCrossPlay** for your server.
